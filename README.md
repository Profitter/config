# Richard's Private  Configuration Vault

## Description
This repo stores my private configurations files for my self-hosted services and used softwares, including configs of proxy tools (Quantumult, Clash Party, Surge) and docker-compose files.

This repo is organised in the form of structured folder named by their usage. 

> **Note**:
>  - This repo is fully private as it contains sensitive information such as subscription link from proxy providers, api key from token providers, etc;
>
>  - This repo is also used as a practice repo for me to get familiar with the usage of git tools and other knowledge required to become a qualified technical writer.

## Guide
The configurations files are stored structurally into the following folders representing their corresponding services:
- [Clash Party](https://clashparty.org/)
- [Quantumult X](https://github.com/crossutility/Quantumult-X)
- [Surge](https://nssurge.com/)
